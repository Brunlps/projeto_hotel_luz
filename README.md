# Sistema de Gerenciamento de Hotel

## O que o sistema faz
Sistema de gerenciamento hoteleiro via terminal com:
- Cadastro de clientes
- Cadastro e consulta de quartos
- Criação, modificação e cancelamento de reservas
- Consulta de reservas e clientes

## Como rodar
python main.py

## Stack
Python 3.x
